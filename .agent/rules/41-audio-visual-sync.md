# Audio-Visual Sync & Editing Guide

Suno V5で生成された楽曲と、Vidu Q2の映像を完璧に同期させるためのガイドライン。

## 1. 構成の分析 (Structure Analysis)
編集前に、音声波形を見てマーカーを打つ。

- **Beat Markers**: キックドラムやスネアに合わせてマーカーを打つ。
- **Section Markers**: Aメロ、サビの変わり目に色付きマーカーを打つ。

## 2. BPMとカット割り (BPM & Cutting)
BPM（Tempo）によって、最適なカットの長さが決まる。

| BPM | 1小節の長さ | 推奨カット長 (Vidu) |
| :--- | :--- | :--- |
| **80** (Slow) | 3.00s | 3.0s (Extend to 6s) |
| **120** (Mid) | 2.00s | 4.0s (2小節) |
| **160** (Fast) | 1.50s | 3.0s (2小節) |

**Vidu Tips:**
- Viduは通常 **4秒** 生成。編集で **2秒** や **3秒** にトリミングして使うのが最も品質が良い（生成直後と直前の動きが安定するため）。

## 3. リップシンク (Lip Sync)
Vidu Q2は、Reference Imageと音声を入力することで、口の動きを合わせる機能（Web UI等の一部機能）を持つ場合があるが、基本は**「雰囲気シンク」**を目指す。

- **完全シンク**: 専用ツール（SadTalker, Haloなど）を併用する。
- **Vidu任せ**: プロンプトに `singing` を入れ、口をパクパクさせるタイミングをサビに合わせる。

## 4. トランジション (Transitions)
- **Cut**: 基本的な切り替え。ビートに合わせてパキッと切り替える。
- **Dissolve**: バラードなどで、前のカットの残像を残しながら切り替える。
- **Match Cut**: Viduのリファレンス機能を生かし、同じキャラクターが「振り返る」動作の終わりと、「走り出す」動作の始まりを繋ぐ。

## 5. エフェクト (Effects)
Sunoの楽器に合わせてエフェクトを乗せる。
- **Kick**: 画面振動（Camera Shake）。
- **Cymbal**: フラッシュ（White Flash）またはグロー（Glow）。
