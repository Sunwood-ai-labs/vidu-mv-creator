# 技術スタック (Technology Stack) for Vidu MV Creator

## Core AI Model
- **Vidu Q2**: Primary Video Generation Model
  - Version: Q2 (Latest)
  - Feature: Reference-to-Video (Character Consistency)
  - Feature: First-to-Last Frame (Loop/Transition)
  - Feature: AI Sound Effects

## Development & Automation
- **Language**: Python 3.10+ (for automation scripts)
- **Video Processing**: FFmpeg (for stitching and format conversion)
- **Image Generation**: Midjourney / Stable Diffusion (for Reference Image creation)

## File Formats
- **Input Images**: PNG/JPG (High Resolution recommended)
- **Output Video**: MP4 (H.264), 1080p
